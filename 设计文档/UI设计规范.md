# Modbus 模拟器 UI 设计规范

## 一、设计定位

| 项目 | 决策 |
|------|------|
| 应用类型 | 工业工具（桌面端） |
| 目标用户 | 工程师、调试人员 |
| 整体风格 | 深色工业风（Dark Mode） |
| UI 框架 | **HandyControl**（NuGet: `HandyControl`） |
| 字体 | Inter（标题 + 正文统一使用） |

---

## 二、UI 框架接入

### 安装

```bash
dotnet add package HandyControl
```

### App.xaml 引入深色主题

```xml
<Application.Resources>
    <ResourceDictionary>
        <ResourceDictionary.MergedDictionaries>
            <hc:ThemeResources RequestedTheme="Dark"/>
            <hc:Theme/>
        </ResourceDictionary.MergedDictionaries>
    </ResourceDictionary>
</Application.Resources>
```

---

## 三、配色方案

| Token 名称 | 色值 | 用途 |
|-----------|------|------|
| `Background` | `#0F1117` | 窗口背景 |
| `Surface` | `#1C2033` | 面板 / 卡片背景 |
| `Primary` | `#2563EB` | 主按钮、Tab 激活、高亮 |
| `Success` | `#22C55E` | 运行中 / 已连接状态 |
| `Warning` | `#F97316` | 轮询中 / 警告状态 |
| `Error` | `#EF4444` | 连接失败 / 错误 |
| `TextPrimary` | `#F1F5F9` | 主文字 |
| `TextSecondary` | `#94A3B8` | 辅助文字、标签 |
| `Border` | `#334155` | 分割线、边框 |

### WPF 资源字典定义示例

```xml
<!-- Resources/Colors.xaml -->
<ResourceDictionary>
    <SolidColorBrush x:Key="AppBackground"   Color="#0F1117"/>
    <SolidColorBrush x:Key="AppSurface"      Color="#1C2033"/>
    <SolidColorBrush x:Key="AppPrimary"      Color="#2563EB"/>
    <SolidColorBrush x:Key="AppSuccess"      Color="#22C55E"/>
    <SolidColorBrush x:Key="AppWarning"      Color="#F97316"/>
    <SolidColorBrush x:Key="AppError"        Color="#EF4444"/>
    <SolidColorBrush x:Key="AppTextPrimary"  Color="#F1F5F9"/>
    <SolidColorBrush x:Key="AppTextSecondary" Color="#94A3B8"/>
    <SolidColorBrush x:Key="AppBorder"       Color="#334155"/>
</ResourceDictionary>
```

## 五、HandyControl 控件选型

| UI 元素 | 推荐控件 |
|--------|---------|
| 模式切换 | `RadioButton` + `ContentPresenter` |
| 协议切换（TCP/RTU） | `hc:ToggleBlock` 或 `RadioButton` |
| 文本输入框 | `hc:TextBox` |
| 端口 / 地址数值输入 | `hc:NumericUpDown` |
| 串口 / 波特率下拉 | `ComboBox`（hc 样式） |
| 启动 / 停止按钮 | `hc:Button` （`Style="{StaticResource ButtonPrimary/Danger}"）` |
| 寄存器数据表格 | `hc:DataGrid` |
| 运行状态指示 | `Ellipse`（绑定颜色）+ `TextBlock` |
| 通知 / 错误提示 | `hc:Growl`（Toast 通知） |
| 分组标题 | `hc:Divider` |

## 六、状态颜色映射

| 状态 | 颜色 Token | 说明 |
|------|-----------|------|
| 已停止 | `AppTextSecondary (#94A3B8)` | 灰色，服务未启动 |
| 监听中 / 运行中 | `AppSuccess (#22C55E)` | 绿色，正常运行 |
| 轮询中 / 连接中 | `AppWarning (#F97316)` | 橙色，进行中 |
| 连接失败 / 错误 | `AppError (#EF4444)` | 红色，异常 |

---

## 七、布局约定

- 内容区内边距：`16px`
- 控件间距：`8px`（同组）/ `16px`（跨组）
- 标签宽度：统一 `80px` 右对齐，与输入框对齐
- 按钮高度：`36px`，最小宽度 `100px`
- 数据表格行高：`32px`
